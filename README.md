# AI IT Helpdesk Assistant

This project is an AI-powered IT support assistant that provides real-time troubleshooting for common technical issues.

## Features
- Chat-based interface using Streamlit
- AI-generated step-by-step solutions
- Handles network, hardware, and software issues

## Tech Stack
- Python
- Streamlit
- OpenAI API

## How to Run
1. Install dependencies:
   pip install streamlit openai

2. Set your API key:
   export OPENAI_API_KEY="your-key"

3. Run the app:
   streamlit run app.py
